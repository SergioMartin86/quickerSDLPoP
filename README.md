quickerSDLPoP
----------------

[![Build & Tests](https://github.com/SergioMartin86/quickerSDLPoP/actions/workflows/make.yml/badge.svg)](https://github.com/SergioMartin86/SDLPoP/actions/workflows/make.yml)

quickerSDLPoP is an attempt to improving the performance of [SDLPoP](https://github.com/NagyD/SDLPoP). 

The main aim is to improve the performance of headless re-recording for TASing and botting (See: [JaffarPlus](https://github.com/SergioMartin86/jaffarPlus)) purposes.
